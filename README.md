# Ejecutar en Desarrollo

1. Clonar el repositorio
2. Ejecutar
```
yarn install
```

3. Tener NestCLI instalado
```
nest i -g @nestjs/cli
```

4. Levantar la base de datos (tener instalado dockerDesktop)
```
docker-compose up -d
```

## Stack usado
* MongoDB
* Nest
* Docker
